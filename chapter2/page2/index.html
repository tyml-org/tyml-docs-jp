    or型🔗

    TYMLでは、どちらかの型を許可したいというケースにも対応できるように設計されています

    /// int型もしくはstring型の値を許可する
setting: int | string
    • TOMLの場合
    # !tyml example.tyml
setting = 100

    # !tyml example.tyml
setting = "string"

    どちらも有効です
